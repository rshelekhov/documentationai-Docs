## Overview

To start, sign up and create your first documentation project.

## Steps

1. Go to `https://rshelekhov.com`
2. Click **Sign Up** and create your account
3. Click "New Project" to begin organizing docs
4. Add pages or import markdown files
5. Invite collaborators if needed

## Tip

All changes are saved automatically as you edit.
