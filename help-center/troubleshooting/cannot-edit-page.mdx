---
title: Cannot Edit Page
description: Fix issues editing a doc page in rshelekhov.
---

## Symptoms

- Edit button is missing
- Cannot save changes to a page

## Causes

- No edit permissions
- Page is locked by another user

## Solution

<Steps>
  <Step title="Check permissions">
    Ask the project owner to grant you edit rights.
  </Step>
  <Step title="Check for locks">
    If a page is locked, wait or contact the user editing it.
  </Step>
</Steps>
